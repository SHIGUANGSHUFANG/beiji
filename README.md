# 📚 背记系统（Memory Learning System）

基于 **Streamlit** 构建的三步骤知识背记 Web 应用，支持上传 Word 文档自动解析知识点与题目。

---

## 功能概览

| 步骤 | 名称 | 说明 |
|------|------|------|
| 1 | 👁️ 知识展示 | 直接阅读原文，填空词橙色下划线高亮，用于初步记忆 |
| 2 | 🫧 气泡池填空 | 根据带空格的原文，从词语气泡池中选择正确词语填入 |
| 3 | ⚡ 快速练习 | 逐题单选，即时显示正误与正确答案 |

### 章节筛选
侧边栏可选择章节，三个步骤均按当前选中章节过滤内容。

---

## 快速开始

### 环境要求
- Python 3.9+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动应用

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`

## Word 文档格式规范

应用通过以下规则自动识别内容类型：

### 知识段落
- Heading 3 = 大节标题（如"一、光的直线传播"）
- Heading 4/5 = 小节标题（如"1. 现象"、"2.小孔成像"）
- Normal 段落 = 知识正文

### 填空词标记
Word 文档中使用**下划线格式**的词语自动识别为填空词：
- 在 Word 中选中词语 → 点击"下划线"按钮（U）
- 解析时自动提取为填空答案

也兼容纯文本格式：
- `___词语___`（前后各3个下划线）

**示例：**
> 成像特点：成<u>倒立</u>的<u>实像</u>；形状与原物相同，与小孔形状<u>无关</u>

### 选择题格式（两种方式均支持）

#### 方式一：背记系统格式（推荐）
```
【第一题】：
【题目内容】：物理学中引入"光线"来描述光的传播，主要采用了哪种研究问题的科学方法？
【正确选项】：运用了构建理想化物理模型的方法
【错误选项1】：主要采用了控制变量与多次实验的方法
【错误选项2】：通过等效替代实际光束的方法来简化
【错误选项3】：它属于类比法，将光与水波类比而建立
```

- 系统自动将【正确选项】设为 A，三个【错误选项】设为 B/C/D
- 练习时选项顺序不变，答案始终为 A

#### 方式二：传统 A./B./C./D. 格式
```
下列说法正确的是？
A. 选项一
B. 选项二
C. 选项三
D. 选项四
【答案】B
```

### 章节分隔
- `📝 练习题` 标记知识区域结束、练习题区域开始
- `========================================` 分隔不同大节

## 部署到 Streamlit Cloud

1. 将本仓库推送到 GitHub
2. 访问 [share.streamlit.io](https://share.streamlit.io)
3. 连接仓库，选择 `app.py` 为入口文件
4. 点击 Deploy，即可获得在线访问链接
